# Word <-> Markdown 双向转换工具

## 功能简介

Word 文档（.docx）与 Markdown（.md）双向互转，便于 AI 大模型（如 Claude、ChatGPT）高效读取，以及将 AI 处理后的 md 文件还原为 Word 文档。

### 核心特性
- docx -> md：Word 转 Markdown，供 AI 读取分析
- md -> docx：Markdown 转 Word，还原为正式文档
- 保留标题层级（Heading 1-6）
- 保留表格结构（双向）
- 保留加粗、斜体等格式
- 支持批量转换（自动识别方向）
- 支持文件夹自动监听

## 快速开始

### 第一步：安装 Python（如已安装可跳过）

1. 访问 [python.org](https://www.python.org/downloads/)
2. 下载 Python 3.8+ 版本
3. 安装时 **勾选 "Add Python to PATH"**

### 第二步：安装依赖

打开命令行（cmd 或 PowerShell），进入本目录运行：

```bash
pip install -r requirements.txt
```

或者：

```bash
python -m pip install python-docx mammoth
```

### 第三步：开始使用

#### 方式一：拖拽转换（最简单）

将 `.docx` 或 `.md` 文件直接 **拖拽** 到 `拖拽转换.bat` 图标上：
- 拖入 `.docx` -> 自动生成 `.md`
- 拖入 `.md` -> 自动生成 `.docx`

#### 方式二：命令行转换

```bash
# docx 转 md
python convert_docx.py 起诉状.docx

# md 转 docx
python convert_docx.py 法律意见书.md

# 批量混转（自动识别方向）
python convert_docx.py a.docx b.md c.docx

# 指定输出目录
python convert_docx.py 文件.docx -o ./output/
```

## 高级功能

### 文件夹监听模式

设置监听文件夹，放入 .docx 或 .md 文件即自动转换：

```bash
# 监听当前目录
python convert_docx.py --watch .

# 监听指定目录
python convert_docx.py --watch "D:\待转换文件"

# 转换后自动删除原文件
python convert_docx.py --watch . --delete

# 调整扫描间隔（默认 3 秒）
python convert_docx.py --watch . --interval 5
```

按 `Ctrl+C` 停止监听。

### 提取文档中的图片（仅 docx->md）

```bash
python convert_docx.py 文件.docx --extract-images
```

图片保存到 `文件名_images/` 文件夹中。

### 强制指定转换引擎（仅 docx->md）

```bash
# pandoc（需先安装，质量最佳）
python convert_docx.py 文件.docx --engine pandoc

# python-docx（含表格支持，默认推荐）
python convert_docx.py 文件.docx --engine python-docx

# mammoth（纯 Python 备选）
python convert_docx.py 文件.docx --engine mammoth
```

---

## 转换引擎说明

| 引擎 | 方向 | 表格 | 格式 | 推荐度 |
|------|------|------|------|--------|
| python-docx | docx->md / md->docx | [OK] | [OK] | *** 默认推荐 |
| pandoc | docx->md | [OK] | [OK] | ** 最佳质量 |
| mammoth | docx->md | 基础 | 良好 | * 备选 |

> docx->md 自动选择引擎，优先级：pandoc > python-docx > mammoth
> md->docx 使用 python-docx

## 适用场景

- **法律文件分析**：将起诉状、判决书、合同等转为 md，供 AI 分析
- **AI 处理后还原**：AI 处理完的 md 文件，一键还原为 Word 正式文档
- **文档归档**：批量将 Word 文档转为易于版本管理的 Markdown 格式
- **知识库构建**：将 Word 格式的知识文档转为 AI 可读的格式

## 常见问题

**Q: 提示 "No module named 'docx'"？**
A: 运行 `pip install -r requirements.txt` 安装依赖。

**Q: 转换后的表格格式不对？**
A: 推荐安装 pandoc（从 [pandoc.org](https://pandoc.org/installing.html) 下载），pandoc 对表格的处理最完善。

**Q: 可以转换 .doc 文件吗？**
A: 支持 .docx 格式（Office 2007+）。旧版 .doc 文件请先用 Word 另存为 .docx 格式。

**Q: 图片会丢失吗？**
A: 默认图片以占位符保留。docx->md 时使用 `--extract-images` 参数可提取图片到文件夹。

**Q: md 转 docx 支持哪些格式？**
A: 支持标题（H1-H6）、表格、粗体、斜体、粗斜体、有序/无序列表、段落、分隔线。
